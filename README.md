# Chat App

## Project Overview
A real-time chat application enabling seamless messaging with features like user authentication, message persistence, group chats, and a responsive user interface.

## Features
- User registration and secure login
- One-on-one and group chat support
- Real-time message exchange using WebSockets
- Message history saved in the database
- Emoji and file sharing
- Responsive design for desktop and mobile
